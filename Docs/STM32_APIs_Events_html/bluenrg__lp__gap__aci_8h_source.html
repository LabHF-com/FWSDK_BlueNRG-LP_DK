aci_gap_clear_security_db
tBleStatus aci_gap_clear_security_db(void)
Clear the security database. All the devices in the security database will be removed. ATTENTION: It is strongly recommended not to give this command during intense radio activity (e.g. during advertising or connection with short intervals, i.e. less than 30 ms, or during scanning), since it will trigger an erase of a Flash sector. After this command, all devices previously recorded in the bonding table and connected when command has been submitted will remain connected, preserving authentication and encryption of the link.
