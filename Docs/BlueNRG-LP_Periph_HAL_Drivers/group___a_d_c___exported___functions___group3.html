HAL_StatusTypeDef HAL_ADC_AnalogWDGConfig ( ADC_HandleTypeDef *  hadc,
const ADC_AnalogWDGConfTypeDef *  AnalogWDGConfig 
)

Configure the watchdog.

Note
Possibility to update parameters on the fly: This function initializes the watchdog, successive calls to this function can be used to reconfigure some parameters of structure "ADC_AnalogWDGConfTypeDef" on the fly, without resetting the ADC. The setting of these parameters is conditioned to ADC state.
