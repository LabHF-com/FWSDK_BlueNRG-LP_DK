uint16_t conn_cid_t_s::Local_CID

Array of up to 5 two-octet values that represent the local channel endpoints. When receiving an event with Event_Type = 0x00 (L2CAP_CFC_CONN_REQ) or 0x02 (L2CAP_ECFC_CONN_REQ), all CID values are equal to 0x0000. On the Requester side (Channel Initiator) receiving an event with Event_Type = 0x01 (L2CAP_ECFC_CONN_RSP), a value equal to 0x0000 means that the requested channel has not been created. Values:

  • 0x0040 ... 0x007F
